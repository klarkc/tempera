# Usage

<!-- usage -->

```sh-session
$ npm install -g tempera
$ tempera COMMAND
running command...
$ tempera (-v|--version|version)
tempera/0.0.0 darwin-x64 node-v14.15.0
$ tempera --help [COMMAND]
USAGE
  $ tempera COMMAND
...
```
